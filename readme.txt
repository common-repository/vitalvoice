== Description ==

С помощью данного плагина вы сможете автоматически озвучивать любые записи на своем сайте. Устанавливаете плагин, активируете его - все! Теперь любой пост, отмеченный "для озвучивания", будет автоматически озвучен одним из 5 голосов профессиональных дикторов. Представьте под каждой из ваших записей появиться стильный плеер, позволяющий слушать ваши посты. Создайте свой подкастинг сегодня!

* Author: [ilyushin-m](http://vitalvoice.ru)
* Project URI: <http://vitalvoice.ru>

This plugin allows you to voice any post with one of five speech synthesis voices. After post publication the player will appear under the written text and will allow your site guests to listen to your post. More information you can find at site vitalvoice.ru or by mail vitalvoice@speechpro.com.

== Installation ==

1. Upload `vitalvoice.inc.php`, `minixml.inc.php`, `vitalvoice.php` and directory `images` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Get API-key on your account on vitalvoice.ru

1. Загрузите файлы `vitalvoice.inc.php`, `minixml.inc.php`, `vitalvoice.php` и папку `images` в `/wp-content/plugins/`
2. Активируйте плагин
3. Получите API-ключ на сайте vitalvoice.ru

или

1. Загрузите zip архив с этой страницы
2. Укажите путь к архиву в "установке плагинов" в личном кабинете вашего сайта
3. Получите API-ключ на сайте vitalvoice.ru

== Frequently Asked Questions ==

= Плеер появился, но звука нет =

Возможно вы не зарегистрировались на vitalvoice.ru и не получили API-ключ.
Возможно ваша учетная запись не активирована. Напишите администратору vitalvoice@speechpro.com
Возможно на вашем тарифе закончились доступные сообщения. Выберете другой тариф.

= Плагин установил, но плееры под постами не появляются =

Возможно вы не активировали плагин или не указали API ключ.
Возможно при публикации поста вы не поставили галочку "Check this option to hear this post".
